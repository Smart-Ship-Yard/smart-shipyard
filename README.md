## 프로젝트 소개
2026 한이음 ICT 멘토링 (해운물류 트랙) 프로젝트임.

조선소 현장이 너무 넓어 관리자가 일일이 돌아다니며 확인하기 힘들다는 점에 착안함. 이동형 로봇(RC카)과 웹캠으로 현장 영상을 실시간 수집하고, 이를 AI 비전으로 분석하여 웹상의 3D 가상 조선소(디지털 트윈)에 띄워주는 시스템임. 관리자가 현장에 가지 않아도 원격으로 바로 상황을 파악하고 대처하는 것이 최종 목표임. 프로젝트의 설계도와 베이스캠프 역할을 함.

## 파트별 레포지토리 안내
본 Main 레포지토리는 프로젝트의 전체 기획, 아키텍처 다이어그램, 회의록 및 전체 시스템 실행 환경(Docker)을 관리함. 각 도메인별 실제 개발 소스코드는 아래의 분리된 레포지토리에서 관리함.

- [Edge (현장 데이터 수집)](https://github.com/Smart-Ship-Yard/Edge): RC카 주행 제어 및 웹캠 실시간 RTSP 송출
- [Backend & AI (서버 및 비전 분석)](https://github.com/Smart-Ship-Yard/Backend-AI): RTSP 디코딩, YOLOv8/Pose 추론, WebSocket 통신, DB 저장
- [Frontend (3D 트윈 대시보드)](https://github.com/Smart-Ship-Yard/Frontend): React/Three.js 기반 3D 렌더링, 실시간 Red Alert 팝업

## 주요 기능
- 안전 관리: 작업자 안전모 미착용, 위험구역 접근, 쓰러짐(낙상) 실시간 감지
- 공정 확인: 선박 블록 조립 진행 상황 시각화
- 품질 및 화재 감시: 용접 불량(스패터)이나 초기 연기, 불꽃 탐지
